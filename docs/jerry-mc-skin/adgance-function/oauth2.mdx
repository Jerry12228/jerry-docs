# OAuth 2

## 创建 OAuth 2 应用

前往[「用户中心」](https://account.mc.jerry.ink/user)->[**「高级功能->OAuth 2 应用」**](https://account.mc.jerry.ink/user/oauth/manage)页面  
点击 **「创建应用」** 填写用户名和回调地址，确认无误后点击确定
