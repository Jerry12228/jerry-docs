# 积分系统

站内诸多操作都需要使用积分进行

:::caution 注意  
文档内容可能更新不及时，具体数值请以操作提示为准  
若您发现本文档有误，请务必及时联系站长，感激不尽  
:::  

:::tip  
文档中粗斜体的部分为您可以获得积分
:::

## 创建/删除角色  
每创建一个角色将会消耗1000积分（删除角色返还）

## 上传材质  
### 材质  
***1. 上传一个公开材质奖励20积分，将公开材质改为私有或删除将扣除20积分，同时补扣相应存储积分***  
2. 当公开材质被其他用户添加到衣柜时每个添加奖励5积分  
3. 上传一个私有材质无积分变动  
***4. 当私有材质被改为公开材质，奖励20积分，返还多扣的存储积分***  
### 材质存储  
1. 上传公开材质消耗 1积分/KB  
2. 上传私有材质消耗 10积分/KB  

## 添加材质到衣柜  
每个材质消耗5积分，从衣柜中移除后返还

## 举报材质  
***每个通过的举报奖励 100 积分***

## 签到  
***每天零点后可签到，每天只能签到一次***  
***每次其签到随机获得 10~20 积分***  

## 正版绑定
***成功绑定正版奖励1000积分***
