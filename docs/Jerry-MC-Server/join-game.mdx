# 加入游戏  

:::tip

在此之前，您需要在[Jerry - MC皮肤站](https://skin.mc.jerry.ink/) [**注册用户**](https://skin.mc.jerry.ink/auth/register) 并 [**创建角色**](../jerry-mc-skin/newbee-guide/add-player)

:::

:::info  
*服务器目前支持持Java版本*  
相关资源已经整理在[**这里**](https://pan.jerry.ink/Minecraft)(人工搬运，更新缓慢但下载超快哦)  
直通车:  
[Java](https://pan.jerry.ink/Minecraft/Java/JDK) \ 
[HMCL](https://pan.jerry.ink/Minecraft/Java/%E5%90%AF%E5%8A%A8%E5%99%A8/HMCL) \

[专用整合包(旧版本无法加入服务器,请及时更新)](https://pan.jerry.ink/public/Minecraft/Scapegoat%20Client/Scapegoat-latest.zip) \ 
:::
