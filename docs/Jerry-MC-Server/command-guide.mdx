无

<BrowserWindow>

## 发私信  
`/msg <player> <massage>`  
### 功能  
通过该指令发送私信  
:::tip  
即便如此，服务器日志还是会记录一切，管理员不保证您的消息不泄露
:::  
### 参数  
player：发消息给该玩家
massage：要发送的消息

</BrowserWindow>
<BrowserWindow>

## 申请传送到玩家  
`/tpa <player>`  
### 功能  
请求传送到某玩家  
### 参数  
player：请求传送到该玩家  

</BrowserWindow>
<BrowserWindow>

## 申请某玩家传送到你  
`/tpahere <player>`  
### 功能  
请求某玩家传送到你这里  
### 参数  
player：被请求的玩家

</BrowserWindow>
<BrowserWindow>

## 取消传送请求  
`/tpacanel`  
### 功能  
取消所有已发出的传送请求

无

</BrowserWindow>
<BrowserWindow>

## 同意传送请求  
`/tpaccept`  
### 功能  
同意其他玩家的传送请求（包括同意tpa和tpahere）  
### 参数  
无

</BrowserWindow>
<BrowserWindow>

## 拒绝传送请求  
`/tpdeny`  
### 拒绝其他玩家的传送请求  
拒绝其他玩家的传送请求（包括tps和tpahere）  
### 参数  
无

</BrowserWindow>
<BrowserWindow>

## 传送到传送点  
`/warp <pagenumber|warp>`  

[**传送点列表**](warp-list)  
### 功能  
查看传送点列表或传送到某个传送点  
### 参数  
#### 查看传送点列表  
pagenumber：查看列表的第几页  
#### 传送到某传送点  
warp：传送点名称（名称太长？别骂了可以敲几个几个字母然后鼠标点击或按Tab键补全）

</BrowserWindow>
