## 正确提问  
正确提问是一种智慧，可以提高效率，节省您的时间和生命

1. 对于文档描述不清的部分，您可以向Jerry反馈，万分感谢  
2. 对于大部分通过百度可以青送解决的问题，不做回答，请您谅解（例：如何下载安装启动器；如何安装游戏、mod）
   同时，若您由于询问此类问题导致被怼，本站概不负责  
3. 提问前，请您确保仔细阅读过本文档，避免提出文档已明确说明的问题  
4. 提问时，请您尽量详细描述您的问题并在必要时附加日志以及配置。支持团队不是神，无法通过
   诸如“我无法启动游戏”，“啊我进不去”，“救命”等只言片语知道网线对面的您到底遇到、经历了什么。  
5. 支持团队智商有限，可能无法解决您的问题，请您海涵。我们将不断努力提升自身水平
